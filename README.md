# Qambranis — Release 1.0

Minimal, image-led website for Qambranis.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

1. Push this folder to a new GitHub repository.
2. Import the repository into Vercel.
3. Add `qambranis.com` and `www.qambranis.com` under Vercel → Domains.
4. Add the DNS records shown by Vercel in Cloudflare.
5. Preserve existing MX/TXT records used for email.

## Release 1 limitations

- Orders are submitted through WhatsApp.
- Online card processing is not activated.
- Product availability is confirmed manually.
- All products display the launch price of PKR 15,000.
